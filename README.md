# Dictionary-App📚 

A simple and fast **Dictionary App** built with **HTML**, **CSS**, and **JavaScript**.  
This app allows users to search for any English word and instantly view its definition, part of speech, example sentence, and pronunciation audio (if available).

It uses the free **DictionaryAPI.dev**, making it perfect for beginners learning API integration and dynamic UI rendering.



## 🚀 Features

### 🔍 Search Features
- Search any English word  
- Fetches real-time definitions  
- Shows part of speech  
- Displays example usage  
- Plays pronunciation audio (if available)

### 🎨 UI Features
- Clean and modern interface  
- Responsive layout  
- Auto-updating result section
